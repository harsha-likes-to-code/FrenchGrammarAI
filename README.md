# FrenchGrammarAI
An AI-powered French grammar correction system built using Hugging Face Transformers and fine-tuned language models.

## 🎯 Project Goal
To build a machine learning model capable of correcting French grammatical errors and eventually deploy it as a chatbot.

## 🧠 Tech Stack
- Python
- PyTorch
- Hugging Face Transformers
- CamemBERT / Seq2Seq models
- VS Code
- Java SpringBoot
- React

## 🚀 Features (Planned)
- Fine-tuned grammar correction model
- Evaluation metrics (BLEU, F1, etc.)
- Chat-style interface
- Future React + Spring Boot integration
